# 문제
- C++ 추론 위해서 필요한 라이브러리들 부분적으로 업그레이드 하다가 버전 충돌 발생
  -> HDMI 연결 모니터 안 나옴, opencv 호출 안됨 등

- GUI 포함되지 않은 opencv 설치 및 Mali X11 드라이버 수동 심볼릭 링크 설정한 뒤 X 서버 깨짐
  -> TTY 콘솔 진입 시도했으나 크래시
  -> Recovery Mode 진입 시도했으나 부트 구조로 인해 진입 안 됨

- FP16, FP32, INT8 rknn 파일들 추론 데이터가 다 필요한 상황인데 구버전 드라이버 등 버전 충돌 문제로 FP16 제외 추론 안 됨
  +) INT8 추론 위해서는 onnx -> rknn 변환 과정에서 양자화 관련 기능 사용 필요한데 M1 보드와 가장 호환 잘 되는 버전인 rknn toolkit2 1.6.0 버전에서는 기능 사용 불가
  --> Odroid M1 보드에서는 FP16을 이용한 추론 및 구현만 가능
